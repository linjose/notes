

整理**適合醫院使用、且符合 ISO 14064-1（組織層級）的溫室氣體（GHG）盤查數據欄位建議**。
依醫院實務（臨床、後勤、設施、採購）調整，可直接用於 **內部盤查、第三方查證、或ESG/永續報告**。

## 二、排放源分類（依 ISO 14064 + GHG Protocol）

### 【Scope 1】直接排放（醫院高度相關）

| 欄位名稱      | 說明                      |
| --------- | ----------------------- |
| 排放類型      | 固定燃燒 / 移動燃燒 / 製程 / 冷媒逸散 |
| 設備/來源名稱   | 鍋爐、柴油發電機、救護車            |
| 燃料/物質種類   | 柴油、天然氣、冷媒R134a          |
| 使用量       | 單位：L、Nm³、kg             |
| 使用期間      |                         |
| 排放係數      | kgCO₂e/單位               |
| 排放量（CO₂e） |                         |
| 資料來源      | 燃料帳單、維修紀錄               |
| 不確定性說明    |                         |

📌 **醫院特有重點**

* 備用發電機
* 救護車與公務車
* 冷凍空調冷媒（MRI、手術室）
