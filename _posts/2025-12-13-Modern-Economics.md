本資料彙整了經濟學各大領域的里程碑理論、主要貢獻者及其核心概念的具體範例。

### 1. 成長與總體理論 (Growth and Macroeconomics)

| 經濟學家 (Scholar) | 理論 (Theory) | 核心概念說明 (Core Concept) |
| :--- | :--- | :--- |
| **羅伯特·梭羅 (Robert Solow)** | 索羅模型 (Solow Growth Model) | 長期經濟成長的主力是**技術進步**，而非單純的資本累積（如加班或增添機器）。 |
| **米爾頓·弗里德曼 (Milton Friedman)** | 恆常所得假說 (Permanent Income Hypothesis) | 消費者主要根據其**長期平均收入**（恆常所得）來調整消費，而非短期或暫時的收入變動。 |
| **芬恩·基德蘭德 (Kydland) 與 愛德華·普雷斯科特 (Prescott)** | 時間一致性問題 (Time Inconsistency Problem) | 政府或央行雖然宣稱會採取嚴厲政策（如升息），但實際執行時因擔心後果（如景氣變差）而軟化，導致**預期政策失效**。 |
| **小羅伯特·盧卡斯 (Robert E. Lucas Jr.)** | 盧卡斯批評 (Lucas Critique) | 當政策發生變化時，經濟個體會改變其預期與行為，使得歷史上觀察到的統計關係（如擴大支出與失業率的關係）**不再適用**。 |



### 2. 一般均衡、福利與貿易 (Equilibrium, Welfare, and Trade)

| 經濟學家 (Scholar) | 理論 (Theory) | 核心概念說明 (Core Concept) |
| :--- | :--- | :--- |
| **肯尼斯·阿羅 (Arrow) 與 傑拉德·德布魯 (Debreu)** | 一般均衡理論 (General Equilibrium Theory) | 在所有市場中，價格與數量最終會同時達到均衡，形成**帕雷托最優**狀態（無法在不損害他人前提下使自己更好）。 |
| **奧林 (Bertil Ohlin)** | 赫克歇爾–奧林模型 (Heckscher–Ohlin Model) | 國家會出口自己**相對豐沛的要素**所密集生產的產品，並進口自己稀缺的要素所密集生產的產品。 |
| **羅伯特·門德爾 (Robert Mundell)** | 最適貨幣區理論 (Optimum Currency Area Theory) | 共同貨幣區要能成功，需要成員國之間有**相似的經濟結構**和**高度的人員/資本流動**，否則單一國家失去調整匯率的能力將導致困境。 |



### 3. 資訊與契約：「講真話」與「好合約」的問題 (Information and Contracts)

| 經濟學家 (Scholar) | 理論 (Theory) | 核心概念說明 (Core Concept) |
| :--- | :--- | :--- |
| **喬治·阿克洛夫 (George Akerlof)** | 檸檬市場 (Market for Lemons) | 由於**賣家擁有私人資訊**，導致買家僅願意支付平均價格，使高品質商品退出市場，造成**逆向選擇 (Adverse Selection)** 和市場效率下降。 |
| **麥可·史彭斯 (Michael Spence)** | 訊號傳遞 (Signaling) | 擁有私人資訊的一方（如高能力求職者）會透過**高成本的行動**（如名校學位、證照）來向市場傳遞真實能力，以區分自己與低能力者。 |
| **約瑟夫·史迪格里茲 (Joseph Stiglitz)** | 篩選 (Screening) | 資訊不足的一方（如保險公司）設計多種合約方案，讓客戶**自行選擇**，從而將不同風險或特性的群體區分出來。 |
| **奧立佛·哈特 (Hart) 與 本特·荷姆斯特羅姆 (Holmström)** | 不完全契約與委託–代理 (Incomplete Contracts, Principal–Agent) | 由於合約無法預見所有情況，因此需要設計**激勵機制**（如績效獎金、股份）來協調委託人（股東）和代理人（經理人）之間的利益，以減少**道德風險 (Moral Hazard)**。 |



### 4. 博弈論與機制設計：互相猜來猜去的世界 (Game Theory and Mechanism Design)

| 經濟學家 (Scholar) | 理論 (Theory) | 核心概念說明 (Core Concept) |
| :--- | :--- | :--- |
| **約翰·納許 (John Nash)** | 納許均衡 (Nash Equilibrium) | 在非合作賽局中，所有參與者都選擇了**給定其他對手選擇下的最佳策略**，且無人有單方面改變策略的動機。 |
| **羅伯特·約翰·奧曼 (Robert J. Aumann)** | 重覆賽局 (Repeated Games) | 當賽局無限期或長期重覆進行時，**未來報復的威脅**可以使參與者之間維持**更具合作性**的結果。 |
| **赫維克茲 (Hurwicz)、馬斯金 (Maskin)、梅爾森 (Myerson)** | 機制設計 (Mechanism Design) | 透過**精心設計的規則**（如拍賣形式），使參與者的**最佳策略就是講出真話**，從而實現社會目標（如資源有效分配）。 |
| **保羅·米爾格龍 (Paul Milgrom) 與 羅伯特·B·威爾遜 (Robert B. Wilson)** | 拍賣理論 (Auction Theory) | 分析不同拍賣規則（如多輪競標）對投標者行為與最終交易結果的影響，目標是優化**資源配置效率**與**賣家收益**。 |



### 5. 行為與實驗：人不是完全理性機器 (Behavioral and Experimental Economics)

| 經濟學家 (Scholar) | 理論 (Theory) | 核心概念說明 (Core Concept) |
| :--- | :--- | :--- |
| **丹尼爾·卡尼曼 (Daniel Kahneman)** | 前景理論 (Prospect Theory) | 人們對於**損失**的感受（損失厭惡）比同等數量的**收益**更為強烈，且決策依賴於**參考點**的設定。 |
| **理查·塞勒 (Richard Thaler)** | 心理帳戶 (Mental Accounting)、助推 (Nudge) | 人們在心中將金錢劃分為不同的**心理帳戶**，影響花費決策。政府或企業可利用**預設選項**等方式溫和引導人們做出更優決策 (Nudge)。 |
| **弗農·史密斯 (Vernon Smith)** | 實驗經濟學 (Experimental Economics) | 在**受控的實驗室環境**中模擬市場機制，觀察參與者的行為和市場結果，以驗證或挑戰經濟理論。 |



### 6. 勞動、市場設計與因果推論 (Labor, Market Design, and Causal Inference)

| 經濟學家 (Scholar) | 理論 (Theory) | 核心概念說明 (Core Concept) |
| :--- | :--- | :--- |
| **大衛·卡德 (David Card)** | 自然實驗 (Natural Experiments) | 利用現實中**類似於隨機分組**的事件（如兩個相似地區只有一個調整政策）來研究政策的**因果效果**，減少統計上的偏差。 |
| **阿爾文·羅思 (Alvin Roth) 與 勞埃德·沙普利 (Lloyd Shapley)** | 穩定匹配 (Stable Matching)、推遲接受演算法 (Deferred Acceptance) | 設計演算法來分配資源（如實習醫師與醫院），以確保最終的**匹配是穩定的**，即不存在一對「彼此都更喜歡對方」而想私下換約的對象。 |
| **約書亞·安格里斯特 (Angrist) 與 吉多·因本斯 (Imbens)** | 因果推論 (Causal Inference) | 應用如**工具變數 (Instrumental Variables)** 等統計方法，利用自然發生的**隨機變動**來分離和估計某一因素對結果的真實、非相關性**因果影響**。 |



### 7. 發展、貧窮與制度 (Development, Poverty, and Institutions)

| 經濟學家 (Scholar) | 理論 (Theory) | 核心概念說明 (Core Concept) |
| :--- | :--- | :--- |
| **阿馬蒂亞·森 (Amartya Sen)** | 能力方法 (Capability Approach) | 衡量發展或貧窮的標準不應僅是收入，而應是人們能擁有的**實質自由與做選擇的能力**（如健康、教育、政治參與）。 |
| **巴納吉 (Banerjee)、杜芙若 (Duflo)、克雷默 (Kremer)** | 隨機對照試驗 (Randomized Controlled Trials, RCTs) | 透過將乾預對象**隨機分組**（一組接受政策，一組不接受），來嚴謹評估政策對貧窮地區的**因果影響**。 |
| **阿傑姆奧盧 (Acemoglu) 與 詹姆士·A·羅賓遜 (Robinson)** | 制度與繁榮 (Institutions and Prosperity) | **包容性制度**（保障產權、法治）與**掠奪性制度**（壟斷、少數人利益）是決定國家長期經濟發展和繁榮的**根本原因**。 |
